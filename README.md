# Slide In on Scroll

## Description

The aim of this exercise is to create a slide in effect for images when scrolling through the page.

## Javascript Used

* querySelectorAll()
* window.scrollY
* window.innerHeight
* element.offsetTop()
* conditionals
* classList()
* scroll event
* debounce function to control scroll event
